🔨 How to Build It Yourself

Step 1: Prerequisites

You need to install:

  • Godot 4.x - Download Here
  • Node.js 18+ - Download Here
  • PostgreSQL 14+ - Download Here (or use Supabase)
  • Git - Download Here

Step 2: Clone the Repository

git clone https://github.com/primora-maker/soul-ascension-game.git cd soul-ascension-game

Step 3: Setup Backend

cd backend npm install cp .env.example .env

Edit .env file with your database credentials:

DATABASE_URL=postgresql://user:password@localhost:5432/soul_ascension NODE_ENV=development PORT=3000 JWT_SECRET=your-secret-key

Step 4: Create Database

npm run migrate

Step 5: Start Backend Server

npm start

You should see:

Soul Ascension Backend Server Running on PORT 3000 Database: Connected

Step 6: Open Game in Godot

  • 1. Open Godot 4.x
  • 2. Click "Import Project"
  • 3. Navigate to: soul-ascension-game/godot/
  • 4. Click "Import & Edit"
  • 5. Wait for resources to load (first time takes ~1 minute)

Step 7: Play the Game

  • In Godot Editor: Press F5 or click the Play button
  • Game launches in editor window
  • You're now playing Soul Ascension!

Step 8: Build for Windows/Mac/Linux

  • 1. In Godot: Project → Export...
  • 2. Select your platform (Windows/Mac/Linux)
  • 3. Click "Export Project"
  • 4. Choose location and click "Save"
  • 5. Your executable is created!

Troubleshooting

  • Backend won't start? Check if port 3000 is available
  • Database error? Verify DATABASE_URL in .env
  • Game won't connect? Make sure backend is running first
  • Godot missing resources? Delete .godot/ folder and restart
